# FindMyRecipe

A Simple app to search for recipes by ingredients.

### Libraries used
1. Android Architecture Components
2. RxJava2
3. Retrofit
4. Glide
5. Dagger2 
6. ButterKnife
7. Robolectric


### Instructions to run the app:
1. Download the code.
2. Open the project in Android Studio 3.0 or later.
3. Let it download artifacts.
4. Run the app.
